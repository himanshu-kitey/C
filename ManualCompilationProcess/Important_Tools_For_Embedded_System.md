GNU Toolchain used in Linux and Embedded Linux development:
-----------------------------------------------------------

- Preprocessor          - cpp
- Compiler              - cc1
- Assembler             - as
- Linker                - ld
- Debugger              - gdb
- Disassembler          - objdump
- Symbol Viewer         - nm
- Link Libraries Viewer - ldd
- Compiler Driver       - gcc
- Editor                - vim
